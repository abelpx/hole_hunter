## 版本规范说明

本项目采用语义化版本规范 (Semantic Versioning 2.0.0)：`v主版本号.次版本号.修订号-预发布标识`

- **主版本号 (Major)**: 不兼容的 API 修改
- **次版本号 (Minor)**: 向下兼容的功能性新增
- **修订号 (Patch)**: 向下兼容的问题修正
- **预发布标识**: alpha (开发中) → beta (测试中) → rc (候选版本) → (正式版)

**版本历程**:
- `v1.0.0-alpha`: 当前开发版本，包含核心功能
- `v1.0.0-beta`: 功能冻结，进入测试阶段
- `v1.0.0-rc`: 候选发布版本
- `v1.0.0`: 首个正式稳定版
