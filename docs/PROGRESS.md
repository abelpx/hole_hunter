## 项目总览

HoleHunter 是一款基于 Nuclei 引擎的现代化 Web 安全测试工具，采用 Electron + React + Go 技术架构。

### P0 - 立即开始

1. **Nuclei 扫描集成**

   - 扫描参数配置
   - 结果解析和入库

2. **扫描进度实时更新**
